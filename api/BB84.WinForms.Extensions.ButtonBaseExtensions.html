WithCommandBinding(ButtonBase, ICommand)

Binds a command to a ButtonBase, enabling the button's click event to execute the command and dynamically updating the button's enabled state based on the command's executability.

public static ButtonBase WithCommandBinding(this ButtonBase buttonBase, ICommand command)

Parameters

buttonBase ButtonBase

The Button to bind the command to.

command ICommand

The ICommand to bind to the button.

Returns

ButtonBase

The ButtonBase control with the binding applied, allowing for method chaining.

Remarks

This method attaches event handlers to the button and the command to ensure that:

  • The button's Click event executes the command when clicked.
  • The button's Enabled property is updated dynamically based on the CanExecute(object) state.
When the button is disposed, the event handlers are automatically detached to prevent memory leaks.
